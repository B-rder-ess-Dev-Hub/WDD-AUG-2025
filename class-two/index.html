Key Accessibility Reminders (a11y)

Accessibility makes the web usable for everyone. Here are the key takeaways from today:

  • Always use a <label> for every <input>. The for attribute on the label must match the id of the input.
  • Write descriptive alt text for all meaningful images. If an image is just for decoration, use an empty alt attribute like alt="".
  • Use placeholders as a hint, not a label. Labels are permanent, but placeholder text disappears when a user starts typing.
